I 路通平台服务-详细接口文档

访问说明
一,以下所有接口参数中,均需要加入其他三项必填参数:
参数	说明
SecurityToken	访问token值,见下面的生成方法
ClientId	访问用户名
Timestamp	当前时间戳,精确到毫秒

二,SecurityToken值的生成方法为（java版）：
long time = System.currentTimeMillis();		//生成当前时间戳
String data = "CHANEL_GD" + time + "222.128.104.218"; //已授权的公网ip
String key = "MGLDSJKLJ780ew54";
String newToken = new String(HMAC.encryptHMAC(data.getBytes(), key), "UTF-8");	//使用HMAC算法加密
System.out.println("时间戳:"+time);
System.out.println("SecurityToken值:"+newToken);

备注:使用HMAC算法进行签名加密，jdk版本1.8+
